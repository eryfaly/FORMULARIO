Java Swing es una herramienta de interfaz gráfica de usuario (GUI) ligera que incluye un amplio conjunto de widgets. Incluye paquete que le permite crear componentes de GUI para sus aplicaciones Java, y es independiente de la plataforma. La biblioteca Swing está construida sobre el conjunto de herramientas de widgets abstractos de Java ( AWT ), un kit de herramientas GUI más antiguo que depende de la plataforma. Puede utilizar los componentes de la GUI de Java como el botón, el cuadro de texto, etc. de la biblioteca y no tiene que crear los componentes desde cero.
